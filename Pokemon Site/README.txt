This program is a Random Pokémon Team Generator.
Clicking on generate, pops up 6 Pokémons and their
moves. If an individual Pokémon is clicked, they
will get locked. So when clicking Generate that
Pokémon will stay while the others are replaced.
Click the Pokémon again to unlock!
Hitting the pokedex button above a Pokemon will show
extra info about a Pokemon. The extra info appears
as a modal pop-up.


This is done by using the PokeAPI.

***********************
Running the program
***********************

- Run app on my port
set FLASK_APP=app.py
flask run --host=0.0.0.0 --port=18139

- Run app local host
set FLASK_APP=app.py
flask run
